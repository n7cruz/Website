This IOS application was a semester long 3 person group assignment with a reduced scope. It was built on XCode with SWIFT, indexed by Elasticsearch, and hosted on an SQL server by AWS. REST APIs were also managed by Elasticsearch, and this entire application served as mostly an exploration for this emergent technology. Elasticsearch is a RESTful search and anlytics engine also capable of being a small data store.
